🚀 MarketMind – Generative AI Powered Sales & Marketing Intelligence Platform

MarketMind is an AI-powered platform designed to help businesses, startups, and marketers make smarter marketing decisions using Generative AI. It provides intelligent insights, automated marketing strategies, and AI-generated content to optimize campaigns and accelerate business growth.

---

🌟 Project Overview

MarketMind combines Artificial Intelligence, data analytics, and marketing intelligence into a single dashboard that enables users to:

• Analyze market trends
• Generate AI-powered marketing strategies
• Create marketing content automatically
• Improve campaign performance with data-driven insights

This platform helps businesses save time, reduce marketing costs, and make smarter strategic decisions.

⚙️ Installation & Setup

Clone the repository:

git clone https://github.com/piduguabhinaya850/marketmind---genai-powered.git

Navigate to the project folder:

cd marketmind---genai-powered

Install dependencies:

npm install

Create environment file:

.env.local

Add your Gemini API key:

GEMINI_API_KEY=your_api_key_here

Run the project locally:

npm run dev

---

🌐 Deployment

This project is deployed using Vercel.

Steps:

1. Push the project to GitHub
2. Import the repository in Vercel
3. Click Deploy

Your app will be live online.
